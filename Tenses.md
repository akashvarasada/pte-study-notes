# Tenses

## Quick Tips & Tricks for PTE Fill in the Blanks

| Rule / Trick                    | Shortcut to Remember                               | Example                                   |
| ------------------------------- | -------------------------------------------------- | ----------------------------------------- |
| **Look for time words**         | Yesterday → Past, Now → Present, Tomorrow → Future | *She ___ yesterday.* → **went**           |
| **Check helping verbs**         | has/have → Perfect, is/am/are → Continuous         | *They have ___ the work.* → **completed** |
| **After “since/for”**           | Usually Present Perfect / Perfect Continuous       | *I have lived here for 5 years.*          |
| **Sequence of events**          | Earlier action → Past Perfect                      | *The train had left before we arrived.*   |
| **Modal verbs**                 | can/will/should + V1                               | *She will ___ tomorrow.* → **come**       |
| **Singular vs plural subject**  | He/She/It → verb+s                                 | *He plays cricket.*                       |
| **Stative verbs**               | know, believe, like usually not continuous         | ❌ *I am knowing* → ✅ *I know*             |
| **If + present, future result** | First Conditional                                  | *If it rains, we will stay home.*         |
| **Passive voice clue**          | be + V3                                            | *The work was completed.*                 |
| **Adverbs clue**                | already, yet, just → Perfect tense                 | *She has already finished.*               |

## Important Tenses for PTE

### 1. Simple Present

| Usage            | Structure          | Example                |
| ---------------- | ------------------ | ---------------------- |
| Facts / routines | Subject + V1(s/es) | She works daily.       |
| Habit            | Subject + V1(s/es) | He drinks tea.         |
| Timetable        | Subject + V1(s/es) | The train leaves at 6. |

| Sentence                        | Answer   |
| ------------------------------- | -------- |
