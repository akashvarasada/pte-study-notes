# Article

## Quick Tips & Tricks

| Rule / Trick                          | Use                      | Example               |
| ------------------------------------- | ------------------------ | --------------------- |
| Use **a** before consonant sound      | singular countable noun  | a book                |
| Use **an** before vowel sound         | starts with vowel sound  | an apple              |
| Sound matters, not spelling           | pronunciation rule       | an hour, a university |
| Use **the** for specific/known things | already identified       | the movie             |
| First mention → a/an                  | non-specific noun        | I saw a dog           |
| Second mention → the                  | specific noun            | The dog barked        |
| No article with plural/general nouns  | general meaning          | Dogs are loyal        |
| No article with uncountable nouns     | abstract/general meaning | Information is useful |
| Use “the” with superlatives           | best, tallest            | the best student      |
| Use “the” with instruments/rivers     | fixed rule               | the piano, the Nile   |

## Main Article Rules with PTE Examples

| Rule                                  | Correct Example                  |
| ------------------------------------- | -------------------------------- |
| Singular countable noun needs article | She bought **a** car.            |
| “An” before vowel sound               | He is **an** honest man.         |
| “A” before consonant sound            | She joined **a** university.     |
| First mention uses a/an               | I watched **a** film yesterday.  |
| Second mention uses the               | **The** film was inspiring.      |
| Unique things take “the”              | **The** sun rises in the east.   |
| Superlatives take “the”               | She is **the** smartest student. |
| Musical instruments use “the”         | He plays **the** guitar.         |
| Ordinal numbers use “the”             | He secured **the** first rank.   |
| General plural nouns take no article  | Teachers need patience.          |

## Last Minute PTE Memory Formula

| Remember                              | Example               |
| ------------------------------------- | --------------------- |
| Sound > Spelling                      | an hour, a university |
| First mention = a/an                  | I saw a cat           |
| Specific mention = the                | The cat was sleeping  |
| Superlative = the                     | the tallest building  |
| General plural = no article           | Cars are expensive    |
| Uncountable noun = usually no article | advice, information   |
| “Yu” sound = use “a”                  | a user, a university  |
| Acronym vowel sound = use “an”        | an MBA, an FIR        |
